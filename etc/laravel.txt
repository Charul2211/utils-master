##What is Laravel Framework?
- Laravel is a free, open-source PHP web framework, created by Taylor Otwell and 
- intended for the development of web applications following the model–view–controller (MVC) architectural pattern.

##What is Laravel Eloquent?
- The Eloquent ORM included with Laravel provides a beautiful, simple ActiveRecord implementation for working with your database.

##What is Luman?
-Lumen is a new project from Laravel creator Taylor Otwell. 
-It's a "micro-framework", meaning it's a smaller, faster, leaner version of a full web application framework. It competes other popular micro-frameworks, Slim and Silex.

##What are Advantages of Laravel?
- Easy and consistent syntax
- Set-up process is easy
- customization process is easy
- code is always regimented with Laravel
